# lodash-rails

[lodash](http://lodash.com/) for the Rails asset pipeline.

## Installation

Add this line to the assets group in your Gemfile:

```ruby
gem 'lodash-rails'
```

Add the necessary library to `app/assets/javascripts/application.js`:

```js
//= require lodash
```

## What's included?

lodash 4.17.21:

- lodash.core.js
- lodash.core.min.js
- lodash.js
- lodash.min.js
